# Project instructions

Use `Roadtripeado_Maps_Guia_PR.docx` as the main content source for this project.

This website should be a modern, stylish, interactive Puerto Rico travel platform focused on family-oriented road trips.

Core feature:
Create a road trip planner that helps users build trips based on:
- categories
- region
- municipio
- easy access
- family-friendly preferences
- one-day or weekend trips
- distance or drive preference

Important:
- Build a polished, mobile-friendly UI
- Organize the guide data into a reusable structured format
- If exact coordinates are missing, use region and municipio logic first
- Keep the architecture ready for future map and geocoding improvements
